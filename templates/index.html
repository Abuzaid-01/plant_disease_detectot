About This Project

How It Works

This Plant Disease Assistant uses an EfficientNet-B2 deep learning model trained on 9 different crops and their associated diseases. The model analyzes leaf images using computer vision techniques to identify patterns associated with different plant diseases.

Technology Stack

Built with PyTorch for the AI model, Flask for the web framework, and Google's Gemini API for generating disease information. The model achieves over 95% accuracy in identifying plant diseases across Apple, Bell Pepper, Cherry, Corn, Grape, Peach, Potato, Strawberry, and Tomato crops.

How To Use

Simply upload a clear image of a plant leaf showing disease symptoms. For best results, ensure the leaf is well-lit and the image is focused. After detection, you can ask the AI assistant specific questions about the disease or request information in your preferred language.

Development

This project combines transfer learning with custom attention mechanisms to improve classification accuracy. The model was trained on thousands of images from the Plant Village dataset, using techniques like data augmentation and label smoothing to enhance performance.
